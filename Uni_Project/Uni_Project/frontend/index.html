How QuantMind Explains Itself

Every allocation is attributed back to its drivers using Shapley-value sampling — the game-theoretic method underlying SHAP. The agent is treated as a function from market features to portfolio weights, and each feature's contribution is measured by how the allocation changes as that feature is revealed across random coalitions.

The chart shows the agent leans most on position inertia (its current holdings — it avoids needless churn), then on MACD trend and RSI signals. No black boxes: when QuantMind recommends an asset, you see exactly which signals moved the decision.
